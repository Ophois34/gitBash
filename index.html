Utilisation de GIT en ligne de commandes

Création d'un dépot

  • Vide: 
    GIT init
  • Par clonage: 
    GIT clone http://github.com/nomUtilisateur/NomDepot

Sauvegarde des modifications (en local)

  • Etat des modifications: 
    GIT status
  • voir les différences entre l'index et le fichier: 
    GIT DIFF
  • Ajout à l'index: 
    GIT add .
  • Enregistrement: 
    GIT commit -M "COMMENTAIRE!!!"
  • Ajout à l'index et commit en même temps: 
    GIT COMMIT -a -m "commentaire"
  • Logs et récupération du N° de dépot: 
    GIT log --oneline
  • Annulation d'un commit: 
    GIT checkout C3B5453 nomFichier

Suppressions

  • Suppression du dernier ajout avant commit: 
    GIT RESET HEAD
  • Supression du dernier commit: 
    GIT RESET HEAD^
  • Suppression du dernier commit pour le fichier: 
    GIT checkout HEAD^ fichier.txt
  • Revient à une version antérieure du fichier
    GIT CHECKOUT 2c3f4567 index.html
  • Annule le dernier checkout: 
    GIT CHECKOUT master index.html

Gestion du dépot distant

  • Si projet non cloné: 
    GIT remote add origin http://github.com/nomUtilisateur/NomDepot
  • Récupération de la version sur le serveur (avant un push): 
    GIT pull
  • Envoi sur le serveur de ce qui a été commité en local: 
    GIT push origin master

Gestion des branches

  • Afficher toutes les branches d'un dépot: 
    GIT branch
  • Créer une branche: 
    Git branch maBranche
  • Se connecter à une branche: 
    GIT checkout maBranche
  • fusionner une branche: 
    GIT merge -ff -m "Intégration de la branche dans la branche courante"
  • Effacer une branche: 
    GIT branch -d maBranche
  • Envoyer sur le serveur dans une branche: 
    GIT push origin maBranche
